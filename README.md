## 🚀 Quick Start

1. Clone this repo (careful, it's hot! 🔥)
   ```
   git clone https://github.com/shiyonvarghese/Portfolio.git
   ```
2. Install the magical dependencies
   ```
   npm install
   ```
3. Launch the rocket
   ```
   npm start
   ```
4. Open your browser and prepare for liftoff at `http://localhost:3000`
